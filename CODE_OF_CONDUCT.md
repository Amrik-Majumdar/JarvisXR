# Code of Conduct

## Our Standard

Project participants must communicate professionally and respectfully. Constructive technical disagreement is welcome. Harassment, discrimination, threats, doxxing, sexualized conduct, deliberate disruption, and publication of another person's private information are not acceptable.

## Scope

This standard applies in repository issues, pull requests, discussions, reviews, and other project spaces.

## Enforcement

Maintainers may edit, remove, reject, or restrict contributions that violate this standard. Serious or repeated violations may result in a temporary or permanent ban from project spaces.

Report conduct concerns privately to the repository owner. Do not include sensitive personal information in a public issue.

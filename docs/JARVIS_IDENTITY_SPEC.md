# JARVIS Identity

## Product Name

The visible product name is **JARVIS**. `JarvisXR` remains the technical project and bundle name.

## Experience

JARVIS is:

- native and portrait-first
- voice-first with a reliable typed fallback
- calm, direct, and action-oriented
- visually centered on one reactive orb
- offline-first where public APIs permit
- explicit about permissions and system boundaries

JARVIS is not:

- a generic chat transcript
- a browser-based shell
- a fake operating-system replacement
- a jailbreak or private-API control layer
- a source of professional or emergency advice

## Interaction States

- Standby: dim and inactive
- Ready: available for input
- Listening: inbound voice capture
- Processing: command interpretation
- Speaking: outbound speech
- Inspection: camera and Vision analysis
- Quiet: speech output reduced or disabled
- Blocked: a clear permission or platform boundary

The app should always distinguish listening from speaking and should state when a feature depends on permission, device support, or an external model.

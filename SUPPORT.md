# Support

JARVIS XR is an open-source prototype maintained on a best-effort basis.

## Before Opening an Issue

1. Read [Installing on iPhone](docs/INSTALLING_ON_IPHONE.md).
2. Check [Troubleshooting](docs/TROUBLESHOOTING.md).
3. Confirm the latest GitHub Actions workflow passed.
4. Reproduce the problem with the latest commit or prerelease.
5. Remove Apple ID details, device identifiers, and personal data from screenshots and logs.

## Where to Ask

- Use a GitHub issue for reproducible bugs or documentation problems.
- Use GitHub Discussions for general setup questions when enabled.
- Use private vulnerability reporting for security problems.

There is no guaranteed response time, service-level agreement, device recovery service, or paid support channel.
